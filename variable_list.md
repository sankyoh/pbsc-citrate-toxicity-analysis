# Variable List

This file lists variables that appear in the analysis do-files: `040_DesStat.do`, `050_RRanalysis.do`, and `051_RRanalysis260606.do`.

## Main variables

| Variable | Description | Used in |
|---|---|---|
| `grade_cat2` | Binary outcome: Grade 2 or higher citrate toxicity. | Risk ratio analysis |
| `combine3` | Three-level electrolyte group: no decrease, either calcium or potassium decrease, or both decreases. | Main risk ratio analysis |
| `i_ca_bin` | Binary indicator for severe hypocalcemia based on minimum ionized calcium. | Descriptive statistics |
| `i_k_bin` | Binary indicator for hypokalemia based on minimum potassium. | Descriptive statistics |
| `i_ca_min` | Minimum ionized calcium level. | Sensitivity analysis |
| `i_k_min` | Minimum potassium level. | Sensitivity analysis |

## Adjustment variables

| Variable | Description | Used in |
|---|---|---|
| `age` | Age. | Descriptive statistics; adjusted models |
| `sex` | Sex. | Descriptive statistics; adjusted models |
| `tbv` | Total blood volume. | Descriptive statistics; adjusted models |

## Output variables generated by regression scripts

| Variable | Description |
|---|---|
| `exposure` | Name of the exposure or comparison. |
| `cRR` | Crude risk ratio. |
| `cCI_l` | Lower bound of the 95% confidence interval for the crude risk ratio. |
| `cCI_u` | Upper bound of the 95% confidence interval for the crude risk ratio. |
| `cPv` | P value for the crude model. |
| `aRR` | Adjusted risk ratio. |
| `aCI_l` | Lower bound of the 95% confidence interval for the adjusted risk ratio. |
| `aCI_u` | Upper bound of the 95% confidence interval for the adjusted risk ratio. |
| `aPv` | P value for the adjusted model. |
| `cRR_CI` | Formatted crude risk ratio and 95% confidence interval. |
| `cP` | Formatted P value for the crude model. |
| `aRR_CI` | Formatted adjusted risk ratio and 95% confidence interval. |
| `aP` | Formatted P value for the adjusted model. |

## Notes

- Patient-level data are not included in this repository.
- Variable definitions are based on the Stata do-files provided in this repository.
- `050_RRanalysis.do` includes variables used in an earlier analysis workflow. Some of these variables are not generated in the current `030_define_vars.do` unless the older commented block is reactivated.
- `051_RRanalysis260606.do` uses `combine3`, `i_ca_min`, and `i_k_min` for the main and sensitivity risk ratio analyses.
